# Node.js Learning Repository

This repository contains multiple Node.js and Express.js projects created while learning backend development.

## Projects

### 1. Airbnb Clone
A simple Airbnb-style application built using Node.js, Express.js, EJS, and Tailwind CSS.

#### Features
- Add Home
- View Homes
- Delete Home
- Dynamic Routing
- MVC Architecture
- EJS Templates
- Tailwind CSS UI

#### Tech Stack
- Node.js
- Express.js
- EJS
- Tailwind CSS

---

### 2. Contact Us Application

- Node.js

- EJS

### 3. Calculator

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm start
```

or

```bash
nodemon app.js
```
